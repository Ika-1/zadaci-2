    $ \frac{x+3}{x-3}+\frac{x-3}{x+3}=3\Leftrightarrow\frac{(x+3)^2+(x-3)^2}{x^2-9}=3\Leftrightarrow\frac{(x+3)^2+(x-3)^2}{x^2-9}-3=0\Leftrightarrow\frac{(x+3)^2+(x-3)^2-3(x^2-9)}{x^2-9}=0 $ $\Leftrightarrow\frac{x^2+6x+9+x^2-6x+9-3x^2+27}{x^2-9}=0\Leftrightarrow\frac{-x^2+45}{x^2-9}=0\Leftrightarrow x^2=45 \wedge x^2\neq 9\Leftrightarrow |x|=\sqrt{45} \wedge |x|\neq 3\Leftrightarrow |x|=3\sqrt{5} \wedge |x|\neq 3$ $\Leftrightarrow \underline{x=-3\sqrt{5}\vee x=3\sqrt{5}}$

решење: Коефицијенти квадратног тринома су $a=5m^4$, $b=-6m^2$ и $c=2$. Први услов, према дефиницији квадратне једначине, је $a\neq 0$, тј. $m\neq 0$, затим, израчунајмо дискриминанту једначине: $\overline{D}=(-6m^2)^2-8\cdot 5m^4=36m^4-40m^4=-4m^4$.
Пошто је $\overline{D}\lt 0$ за све $m\neq 0$, онда су решења једначине увек комплексни бројеви.

решење: Коефицијенти једначине су $a=1$, $b=-7$ и $c=9$. Применом Вијетеових формула $\left\{\begin{array}{c} \alpha + \beta = -\frac{b}{a}\\ \alpha \beta = \frac{c}{a}\end{array}\right\}$ имамо $\left\{\begin{array}{c} \alpha + \beta = 7\\ \alpha \beta = 9\end{array}\right\}$. Затим, $(\alpha - \beta)^2 = \alpha^2 - 2\alpha \beta + \beta^2 = \alpha^2 + 2\alpha \beta + \beta^2 - 4\alpha \beta = (\alpha + \beta)^2 - 4\alpha \beta = 7^2 - 36 \underline{= 13}$.
$\alpha^4 + \beta^4 = \alpha^4 + 2\alpha^2 \beta^2 + \beta^4 - 2\alpha^2 \beta^2 = (\alpha^2 + \beta^2)^2 - 2\alpha^2 \beta^2 = ((\alpha + \beta)^2 - 2\alpha \beta)^2 - 2(\alpha\beta)^2 = (7^2 - 18)^2 - 2\cdot 81 = 31^2 - 162 = 961 - 162 \underline{= 799}$.

решење: $x^2 - \frac{50}{2x^2+7} = 7\Leftrightarrow x^2(2x^2+7)-50=7(2x^2+7) \wedge 2x^2+7\neq 0, $ за све $x\in R$.
$\Leftrightarrow 2x^4+7x^2-50=14x^2+49\Leftrightarrow 2x^4-7x^2-99=0$. За решавање биквадратне једначине уводимо замену $x^2=z$, тако да је $\Leftrightarrow 2z^2-7z-99=0$,
$z_{1,2}=\frac{7\pm\sqrt{49+8\cdot 99}}{4}=\frac{7\pm\sqrt{841}}{4}=\frac{7\pm 29}{4}$, тј. $z=9\vee z=-\frac{11}{2}$. Враћамо се на замену, $x^2=9\vee x^2=-\frac{11}{2}$
$\Leftrightarrow |x|=3\vee |x|=i\sqrt{\frac{11}{2}}\Leftrightarrow \underline{x=3\vee x=-3\vee x=i\sqrt{\frac{11}{2}}\vee x=-i\sqrt{\frac{11}{2}}}$.
